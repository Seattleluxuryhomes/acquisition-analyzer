ACQUISITION ANALYZER — START HERE
=================================

You need: Node.js installed (nodejs.org), your ATTOM key, your Claude key.

----------------------------------------
STEP 1 — Put your keys in a file
----------------------------------------
There's a file called  ENV_FILE_rename_to_.env.txt

a) Open it.
b) Paste your two keys where it says to.
c) Rename the file to exactly:   .env
   (yes, just ".env" — no name in front, no .txt at the end)

It should end up looking like this:

   ATTOM_API_KEY=abc123yourattomkey
   ANTHROPIC_API_KEY=sk-ant-yourclaudekey
   PORT=3000

----------------------------------------
STEP 2 — Install (one time)
----------------------------------------
Open a terminal in this folder and run:

   npm install

----------------------------------------
STEP 3 — Start it
----------------------------------------
   npm start

You'll see:  Acquisition Analyzer on http://localhost:3000

----------------------------------------
STEP 4 — Use it
----------------------------------------
Open your browser to:   http://localhost:3000
Type an address. Click Analyze.

----------------------------------------
Did it work? Check the keys loaded:
----------------------------------------
In a browser go to:  http://localhost:3000/api/health
You want to see:     "attomKey":true   "anthropicKey":true
Two trues = you're good.

----------------------------------------
One safety note
----------------------------------------
If your ATTOM key is the one you typed into chat earlier, make a NEW one
in your ATTOM dashboard and use that instead — the old one isn't safe anymore.
Never put keys inside the code files. Only in .env.
